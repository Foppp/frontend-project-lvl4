# Slack Chat App (simple version)
## Main language RU (EN optional)

* ### React and Redux Toolkit 
* React-router-dom
* React-bootstrap
* Yup
* Socket.io
* i18next
* Formik
* Axios

```sh
$ make install
```

```sh
$ make start
```
